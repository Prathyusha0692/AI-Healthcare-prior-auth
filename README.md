AI-Healthcare-prior-auth

An AI-powered healthcare prior authorization platform that uses a multi-agent workflow to analyze clinical context, detect policy gaps, and generate recommendations.

Overview

This project is designed to help streamline prior authorization review by combining:
- clinical extraction,
- policy analysis,
- gap detection,
- recommendation generation.

It includes a FastAPI backend, Streamlit frontend, and a LangGraph-based agent pipeline.

Features

- Multi-agent orchestration with LangGraph.
- Clinical text processing and analysis.
- Policy and gap detection workflow.
- Recommendation generation.
- FastAPI backend APIs.
- Streamlit frontend interface.
- RAG-style document parsing and retrieval utilities.

## Tech Stack

- Python
- FastAPI
- Streamlit
- LangGraph
- LangChain
- OpenAI API
- Spacy
- AWS-compatible deployment support
